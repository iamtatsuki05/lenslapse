# LensLapse

An in-browser logit lens over training checkpoints: scrub a slider across a model's public
checkpoints and watch per-layer, per-position predictions crystallize from frequency bias into
knowledge — plus a `pip install lenslapse` local probe server for your own models.

- **Live demo**: <https://iamtatsuki05.github.io/lenslapse/>
- **Project page**: <https://iamtatsuki05.github.io/lenslapse/about/>
- **Source**: <https://github.com/iamtatsuki05/lenslapse>
